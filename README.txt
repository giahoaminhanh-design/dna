Cách deploy lên GitHub Pages (repo giahoaminhanh-design/dna):

1) Tải file zip này về, giải nén.
2) Vào repo dna trên GitHub (https://github.com/giahoaminhanh-design/dna).
3) Upload 3 file: index.html, style.css, script.js vào nhánh main (thư mục gốc /).
4) Vào Settings → Pages:
   - Source: Deploy from a branch
   - Branch: main  / (root)
   - Save
5) Trang sẽ có ở https://giahoaminhanh-design.github.io/dna/

Không cần npm hay cài đặt gì thêm.
